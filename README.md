# Keen Keeper

A simple app to help you stay organized and on top of your tasks.
Built as a personal project using Next.js and some cool libraries.

## Built With

- Next.js 16
- React 19
- Tailwind CSS v4
- DaisyUI v5
- Recharts
- React Toastify
- React Icons

## Features

- Clean charts to visualize your data using Recharts
- Smooth page transitions with Framer Motion
- Toast notifications so you always know what's happening
